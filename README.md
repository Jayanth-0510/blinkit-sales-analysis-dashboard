# Blinkit Sales Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes Blinkit's grocery sales data to uncover business insights through interactive visualizations and KPIs.

## Key Metrics

- Total Sales: $1.20M
- Average Sales: $141
- Number of Items: 9K
- Average Rating: 3.9

- Sales Trend Analysis
- Outlet Performance
- Item Type Analysis
